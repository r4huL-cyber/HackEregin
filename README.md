# HackEregin 🔐

HackEregin is a **universal encoding and cipher detection tool** built for CTF players, bug bounty hunters, and cybersecurity researchers.

The tool automatically detects and decodes multiple encoding formats and ciphers.

---

## ⚡ Features

- Automatic encoding detection
- Multiple decoding algorithms
- Hash detection
- Simple terminal interface
- Built for **CTF, bug bounty, and security research**

---

## 🔓 Supported Methods

| Encoding / Cipher | Supported |
|-------------------|-----------|
| Base64 | ✅ |
| Base32 | ✅ |
| Base85 | ✅ |
| Base58 | ✅ |
| Hex | ✅ |
| Binary | ✅ |
| URL Encoding | ✅ |
| HTML Entities | ✅ |
| ROT13 | ✅ |
| ROT5 | ✅ |
| ROT18 | ✅ |
| Atbash Cipher | ✅ |
| Reverse | ✅ |
| Morse Code | ✅ |
| Hash Detection (MD5 / SHA) | ✅ |
